# How To Ask AI

Keep it simple.

## Default command style
```text
Use the project brain and continue: <what I want>
```

## Examples
- `Use the project brain and continue: build user onboarding API.`
- `Use the project brain and continue: review this branch for release risk.`
- `Use the project brain and continue: finish MVP checklist remaining items.`

That is enough. The brain handles startup, process, and documentation discipline.
