# MVP Checklist Template

## Blocking
- [ ] Runtime states canonical and mapped end-to-end
- [ ] Strategy schema canonicalized and validated
- [ ] Event/channel contracts fixed and versioned
- [ ] Determinism rules locked and tested
- [ ] Contract test suite exists
- [ ] MVP out-of-scope boundaries enforced

## High
- [ ] Risk guard minimum rules implemented
- [ ] Error recovery idempotency tested
- [ ] Performance benchmark recorded
- [ ] Gap-recovery behavior defined and tested

## Go/No-Go
Only declare MVP complete when all required items are checked.
