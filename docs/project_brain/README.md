# Project Brain (Template)

## Mission
Persistent memory + workflow framework for AI-assisted software delivery.

## Read order for every new AI session
1. `ENTRYPOINT.md`
2. `PROJECT_CONTEXT.md`
3. `SESSION_STATE.md`
4. `BACKLOG.md`
5. `TEST_GATE.md`
6. `RISK_REGISTER.md`
7. `ENVIRONMENT_BASELINE.md`

## Core files
- `ENTRYPOINT.md`: startup flow for low-context sessions
- `OPERATING_PROTOCOL.md`: strict work protocol
- `SESSION_STATE.md`: current ground truth
- `BACKLOG.md`: prioritized queue
- `ROADMAP.md`: phase view
- `TEST_GATE.md`: quality gate
- `RISK_REGISTER.md`: active risks
- `DECISION_LOG.md`: locked decisions
- `CHANGELOG_AI.md`: session history
- `MVP_CHECKLIST_TRACKER.md`: MVP acceptance gate
- `HANDOFF_TEMPLATE.md`: session handoff format
